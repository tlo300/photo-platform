# Photo Platform

A self-hosted photo and video platform — a privacy-first Google Photos alternative.

Built with FastAPI, PostgreSQL, MinIO, Redis, and Next.js. Runs on Docker Compose locally and on any European VPS.

## Getting started

Copy `.env.example` to `.env` and fill in your values, then:

```
docker compose up
```

See `docs/deploy.md` for full deployment instructions.
